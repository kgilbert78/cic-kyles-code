brief description:

  1. What components/classes you used

    Bootstrap classes: m for margin t for top such as mt-5, p for padding such as pt-2, text size such as fs-3, texct-center, text-uppercase and text-capitalize.
    My own components: margin, text align, font size, color.

  2. If you have any questions or had any trouble

    Is there a way to override bootstrap's text alignment or margin for one portion of text? I suppose the easiest solution is to use bootstrap for these functions throughout.

  3. Anything extra you did that you are proud of/learned

    I was pleased that I thought of inserting a span and assigning a different class to the span, and it worked to override bootstrap classes on the div it was within.
