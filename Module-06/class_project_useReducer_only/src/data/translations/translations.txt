Dafür gibt es aber die Möglichkeit, das Betriebssystem in den Dark Mode zu setzen. Noch steht in der Schwebe, ob der Softwarekonzern auf die dunkle Variante als Standardeinstellung setzt. So oder so. Wir zeigen dir, wie du mit ein paar Klicks zwischen beiden Modi wechseln kannst. Nach diesem Kurztutorial weißt du, wie sich der Dark Mode von Windows 11 aktivieren bzw. deaktivieren lässt.
